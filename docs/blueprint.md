# **App Name**: AI Money Mentor

## Core Features:

- User Authentication & Profile: Secure user registration, login, and personalized profile management for inputting and storing financial data. The data persistence layer will be backed by a relational database to support official project requirements.
- Personalized Onboarding: An intuitive onboarding flow that guides new users to input essential financial details like age, income, expenses, existing investments, and life goals.
- Money Health Score Tool: An AI-powered tool that provides a comprehensive financial wellness score across 6 key dimensions: emergency preparedness, insurance coverage, investment diversification, debt health, tax efficiency, and retirement readiness.
- FIRE Path Planner Tool: An AI-driven tool that generates a month-by-month financial roadmap, including personalized SIP amounts, asset allocation shifts, insurance gap analysis, and emergency fund targets based on user goals (e.g., Financial Independence, Retire Early).
- Tax Optimization Tool: An AI tool that analyzes user's salary structure or uploaded Form 16, models tax liability under old and new regimes, identifies missing deductions, and suggests tax-saving investments tailored to risk profile and liquidity needs.
- AI Guidance Disclaimer: A prominent and clear disclaimer that explicitly states the AI-generated financial guidance is for informational purposes only and does not constitute licensed financial advice, ensuring regulatory compliance.

## Style Guidelines:

- Light color scheme, conveying professionalism and clarity.
- Primary color: Muted deep cyan-blue (#1B6F7D) for trust and stability, reflecting a calm yet knowledgeable financial approach.
- Background color: Very light blue-grey (#ECF2F3) providing a clean, spacious, and unintrusive canvas for content.
- Accent color: Vibrant mint green (#33E6C6) to highlight calls to action and important interactive elements, suggesting growth and clarity without being distracting.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy, and precise feel, suitable for an AI-powered application.
- Body font: 'Inter' (sans-serif) for clear readability and an objective, neutral presentation of financial information.
- Utilize a set of professional, minimalist icons to represent financial concepts, goals, and actions, ensuring clarity and intuitive navigation.
- Initial welcome page with a full-page HD background and a prominent 'Enter' button for aesthetic impact. Overall layout will be clean, responsive, and organized to facilitate easy digestion of complex financial information, emphasizing actionable insights. The application will include a static footer displaying 'All rights reserved Deepak Kumar with current year' to address the copyright requirement.
- Subtle, professional animations and transitions to enhance user interaction, provide feedback on data inputs or calculations, and smoothly guide the user through the financial planning process without overwhelming them.
